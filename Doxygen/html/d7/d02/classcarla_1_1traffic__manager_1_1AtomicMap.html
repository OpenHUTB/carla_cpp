详细描述

template<typename Key, typename Value>
class carla::traffic_manager::AtomicMap< Key, Value >

一个线程安全的、基于unordered_map的原子映射类。

这个模板类提供了一个线程安全的映射，使用mutex来确保对内部unordered_map的并发访问是安全的。

模板参数
Key映射的键类型。
Value映射的值类型。

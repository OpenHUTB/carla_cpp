18 // bSynchronousMode用于控制是否启用同步模式。
19 // 当设置为true时，可能意味着游戏或模拟按照固定的节奏或同步机制来推进帧更新等操作；

26 // 若为false则正常进行渲染操作。

49 // 超出这个距离的物体等可能会被从渲染或其他处理流程中剔除，以节省性能开销，初始值为0.0f表示默认不进行剔除或者还未设置具体的剔除距离。

52 // bDeterministicRagdolls用于控制布娃娃系统是否具有确定性。

54 // 设为false则布娃娃物理表现可能会有更多的随机性等情况。
55 UPROPERTY(EditAnywhere, BlueprintReadWrite)

58 // TileStreamingDistance用于设定瓦片流送的距离，单位是米，这里设置为300000.f也就是3km。
59 // 通常在一些大型场景中，用于控制距离玩家多远的瓦片资源进行加载等流送操作，以平衡性能和场景完整性。
60 float TileStreamingDistance = 300000.f; // 3km
61
62 // ActorActiveDistance表示角色激活的距离，单位是米，此处设为200000.f即2km。

64 float ActorActiveDistance = 200000.f; // 2km
65
66 // SpectatorAsEgo用于确定是否将旁观者（观察者角色）视为自我主体。
